## v0.7 — Goal Engine + 指令系统 + 许知微

日期：2026-06-19

### 新增

- **Goal Engine 模块**：声明式 DSL 条件判定器（`all`/`any`/`not`/`clueKnown`/`factConfirmed`/`eventOccurred`/`stateEquals`），递归可测试，JSON 外置
- **指令系统**：12 条指令，4 分类（information/memory/action/interaction），`command-registry.json` 统一声明，触发词匹配
- **许知微主动引导**：游戏开始后自然入场（立绘动画），三轮渐进引导（介绍目标→轻提示→确认熟练度），对话限制（只解释不执行）
- **指令栏**：底部常驻目标栏 + 指令栏（线索/人物/记忆/帮助），第 4 轮收起
- **目标完成正反馈**：卡片弹出式通知（目标完成 + 新目标提示 + 记忆记录）
- **立绘入场动画**（PortraitIntro）：NPC 头像从全屏缩放到对话框头像位，Web Animations API

### 修复

- 指令结果从隐藏日志抽屉改为可见内容区域渲染
- 许知微欢迎语触发时机从 `init()` 移到游戏开始后 2 秒自然触发
- 场景 JSON 文件改为从独立文件加载

### 已知问题

- Goal Engine 对传入定义对象有副作用（JSON 重新加载时影响低）
- `app.js` 接近 800 行，建议后续拆分
