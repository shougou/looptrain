# LoopTrain 试玩版控制流

## 1. 标准成功路径

```text
探索模式
  ↓ 玩家输入：和小宁对话
对话模式：小宁
  ↓ 玩家温和询问
候选线索：地板下方的滴答声、小宁也听见过声音
  ↓ 结束对话
对话结算卡：获得线索，AP 10 → 7，时间 08:45 → 08:48
  ↓ 探索模式
玩家检查座位下方
  ↓
获得线索：声音不来自座位下方
  ↓
玩家说服赵乘警检查地板
  ↓
试玩版成功
```

## 2. 小宁妈妈隐藏节点

触发条件：

```text
当前 NPC = 小宁
小宁信任 >= 30
玩家询问：妈妈 / 布娃娃 / 一个人 / 害怕
```

效果：

```text
获得线索：小宁妈妈与布娃娃
小宁信任增加
不推进主线
不触发试玩成功
```

## 3. 失败路径

失败条件：

```text
AP <= 0
或 clock >= 09:00
```

失败结算卡显示：

```text
失败原因
本轮确认信息
带入下一轮的记忆
下一轮建议
```

## 4. 下一轮继承

下一轮重置：

```text
mode = explore
clock = 08:45
AP = 10
NPC 状态重置
```

继承：

```text
carried_memory
known_clues 中可继承的线索
```

## 5. 防剧透原则

失败结算只能总结玩家已经获得的信息。
不能直接告诉玩家真正敌人、正式版炸弹位置、隐藏 NPC 的完整真相。
