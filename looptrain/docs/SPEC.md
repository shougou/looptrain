# LoopTrain Standalone 试玩版规格说明

## 1. 产品名称

```text
LoopTrain
```

当前试玩剧集：

```text
第七节车厢：试玩版
```

## 2. 产品定位

LoopTrain 是一个独立互动叙事解谜游戏。

玩家在列车循环中通过：

- 探索
- 对话
- 行动
- 查看线索
- 推动剧情
- 失败循环
- 记忆继承

逐步接近事故真相。

## 3. 当前运行方式

当前本地运行方式：

```bash
bash scripts/start_slt.sh
```

本地入口：

```text
http://127.0.0.1:3030/
```

不依赖 SillyTavern。

## 4. 试玩版背景

```text
1939 年冬，重庆。
渝江线 307 次夜行列车驶向江城。
09:00 前，列车将在北江铁桥前爆炸。
```

试玩目标：

```text
证明第七节车厢存在异常，并说服赵乘警检查地板。
```

## 5. 核心玩法

| 模式 | 说明 |
|---|---|
| 探索模式 | 玩家输入行动，观察车厢、寻找 NPC、检查环境 |
| 对话模式 | 玩家与当前 NPC 交谈，获得线索或触发事件 |

输入通道：

| 通道 | 说明 |
|---|---|
| 扮演 | 自然语言行动/对话 |
| 指令 | 查看线索、查看人物、查看状态、结束对话等明确命令 |

## 6. 核心系统

- AP / 时间推进
- NPC 对话轮数限制
- 线索获取与继承
- 失败结算
- 下一轮记忆继承
- 试玩版成功结算

## 7. 裁判原则

Engine 是唯一裁判。

LLM 不得直接修改：

- AP
- 时间
- 线索
- NPC 状态
- 成功/失败
- 继承记忆

## 8. 当前验收路径

成功路径：

```text
进入第七节车厢
→ 和小宁对话
→ 提到声音
→ 结束对话
→ 说服赵乘警检查地板
→ 试玩版结束
```

失败路径：

```text
强制失败测试
→ 循环失败
→ 进入第 2 轮
```
