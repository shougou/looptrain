# 试玩版设计说明：第七节车厢

## 核心目标

试玩版不解决完整主线，不揭示真正敌人，不要求玩家拆弹。

试玩版只验证一个小闭环：

```text
通过对话获得线索
→ 用线索说服赵乘警
→ 证明第七节车厢存在异常
```

## NPC 功能

### 小宁

情绪入口与第一线索来源。玩家温和询问可获得“地板下方滴答声”。询问妈妈/布娃娃可触发隐藏情感节点。

### 赵乘警

规则门槛。至少需要两条有效证据才会检查地板。

### 沈墨寒

误导与压力。灰大衣纸条让玩家怀疑他，但试玩版不确认其真实立场。

### 小宁妈妈

隐藏 NPC，不实际登场。仅通过小宁的特殊对话触发，不推进环境剧情。

## 成功条件

拥有至少两条有效证据，并触发“说服赵乘警检查地板”。

有效证据包括：

- 地板下方的滴答声
- 小宁也听见过声音
- 声音不来自座位下方
- 连接处有人停留过

## 失败继承

失败后保留已经确认的信息，例如：

- 小宁听见过滴答声
- 赵乘警需要证据才会行动
- 餐车方向的口琴声是疑点

不得继承未发现真相。
