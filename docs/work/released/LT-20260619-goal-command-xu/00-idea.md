# Idea: 行动目标系统 + 指令系统 + 许知微集成

## 背景

v0.6 完成了 Runtime 架构，但玩家缺少目标感和操作引导。需要：
- 让玩家知道"为什么做"和"做完会怎样"
- 让常用操作（查看线索/人物/记忆）不再靠"猜命令"
- 让许知微从代码层面的存在变成游戏里真实可见的同伴

## 目标

- Goal Engine: 声明式 DSL 条件判定，可外置 JSON 配置
- 12 条指令系统: 统一注册、触发词匹配、分类管理
- 许知微: 主动入场引导、三轮渐进提示、只读查询不写入状态
- UX/UI: 目标栏 + 指令栏 + 3 轮学习曲线

## 非目标

- 不自动结算对话（保留玩家手动控制）
- 不做任务列表 UI（避免 RPG 清单感）
- 许知微不替玩家决策或执行指令
