# Game Design

## 概述

《LoopTrain：寒灯初醒》是一款手机端互动叙事解谜游戏。玩家在 1939 年冬的江城号列车上醒来，知道自己 15 分钟后会爆炸，但不记得自己是谁。通过与 NPC 对话、观察环境、收集线索，在循环中逐步揭开真相。

## 核心机制

- **时间循环**: 每轮 14:00→14:15，15 分钟游戏时间
- **AP 行动点**: 每次行动消耗 AP，归零或超时触发失败
- **线索系统**: 8 条核心线索，含 1 条派生线索（3 条满足后生成）
- **目标系统**: 8 个目标分三轮激活，GoalEngine DSL 判定
- **记忆继承**: 失败后保留已确认线索，下一轮可基于记忆推进

## 视觉基调

- 深色背景，列车/隧道/夜间/低照度
- 克制、冷静、悬疑
- 手机竖屏优先，390px 基准
- 不商业官网风格，不过度赛博朋克

## 试玩版结局

爆炸仍发生，但三号车厢已清空。许知微揭示主角代号："寒灯。这不是第一次循环，也不会是最后一次。"
